### 链表篇
- 对应文件夹：`linked_list`

#### 知识点
链表的题通常需要注意两点：
1. 舍得用变量，千万别想着节省变量，否则容易被逻辑绕晕
2. head 有可能需要改动时，先增加一个假head（即哨兵节点sentinel），返回的时候直接取 假head.next，这样就不需要为修改 head 增加一大堆逻辑了。
